# Sociaal Domein Kompas — versie zonder extra accounts

Deze versie gebruikt alleen:
- GitHub
- Vercel

Geen OpenAI-account.
Geen Bing-account.
Geen Python of andere software op je laptop.

## Wat doet de app?

A. Thema's tonen met mogelijke klachten  
B. Cliënt kiest thema  
C. App toont doelen  
D. Cliënt kiest doel  
E. App toont interventies  
F. Cliënt kiest gemeente  
G. App zoekt automatisch via een Vercel API-route  
H. App toont mogelijke opties  

## Uploaden naar GitHub

Upload precies deze onderdelen:

- map `app`
- bestand `package.json`
- bestand `README.md`

Niet de ZIP uploaden.

## Deployen in Vercel

1. Ga naar Vercel.
2. Add New Project.
3. Kies je GitHub repository.
4. Klik Deploy.

Later kunnen we dit verbeteren met OpenAI/Bing API voor betere beoordeling en rangschikking.
