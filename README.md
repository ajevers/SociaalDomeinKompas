# Sociaal Domein Kompas MVP — deel 7

Kleine verbetering:
- De zoekfunctie let niet meer alleen op de exacte interventienaam.
- Per interventie zoekt de app ook op verwante termen.
- Resultaten worden nog steeds alleen getoond als er een concrete bronlink is.

Voorbeeld:
- Mantelzorgcoaching zoekt ook op mantelzorgondersteuning, mantelzorgadvies, mantelzorgconsulent en steunpunt mantelzorg.
- Budgetbeheer zoekt ook op schuldhulpverlening, geldzaken, budgetcoaching en financiële begeleiding.
